# Módulo: Python Intermedio

## 📋 Descripción

Este módulo profundiza en conceptos intermedios de Python, incluyendo programación funcional, manejo avanzado de excepciones, decoradores y generators. Aprenderás técnicas que usan los desarrolladores profesionales.

**Conceptos clave**: Decoradores, generators, context managers, closures, itertools, collections, manejo de excepciones, expresiones regulares.
